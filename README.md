# ID3 决策树实验报告

这是太原理工大学 计算机科学与技术专业 23 届机器学习课程的实验项目。

## 项目内容
- 🌲 核心算法：基于信息熵（Entropy）实现的 ID3 决策树。
- 📊 数据集：在 Iris, Wine, Breast Cancer, Digits 四个经典 UCI 数据集上进行测试。
- 📝 图片：包含可视化图片。

## 环境依赖
运行前请确保安装了必要的库：
```bash
pip install pandas matplotlib scikit-learn seaborn
```
## 🚀 快速开始
克隆仓库并运行主程序：
```bash
python main.py
```
